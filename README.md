# UE4 Battle Royal Tutorial Project

This project is based on a Battle Royal game development tutorial using Unreal Engine 4.18.3.
While following the tutorial as a foundation, I also incorporate ideas and gameplay elements inspired by my own experience playing Battle Royal games such as PUBG. Some systems and mechanics are modified or extended beyond the tutorial to better reflect practical gameplay scenarios and personal design choices.

## Engine

Unreal Engine 4.18.3.

## Description

This project is created for learning Unreal Engine gameplay systems and mechanics by implementing features inspired by Battle Royal games.

## Purpose

- Learn Unreal Engine development workflow
- Practice Blueprint-based gameplay implementation
- Understand core Battle Royal mechanics
- Improve problem-solving skills through game system design
- To gain hands-on experience with Unreal Engine tools and environment setup (textures, foliage, meshes)
- Strengthen C++ proficiency for game development through practical implementation.
